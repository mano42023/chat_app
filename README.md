# chat_app

Clone the repo

## Virtual environment setup
```sh
cd chat_app
python -m venv env
# for window
.\env\Scripts\activate

# For linux
Source env/bin/activate

pip instal -r requirment.txt
```

## Start server
```sh
cd chat_app\server 
python app.py
```

## start UI
```sh
cd chat_app\chat_app_ui
ng serve
```
